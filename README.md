# 📋 FMLA Guide App

A bilingual (English/Spanish) mobile app that helps employees understand 
their rights and responsibilities under the Family and Medical Leave Act (FMLA).

Built for the Social Good Hackathon challenge.

## 📱 Features
- ✅ **Do I Qualify?** — Step-by-step eligibility checker
- 🛡️ **Know Your Rights** — Expandable FMLA rights and responsibilities
- 📄 **Documents Needed** — Interactive checklist based on leave type
- 💬 **Ask AI** — Claude-powered chat assistant for FMLA questions
- 🌐 **Bilingual** — Full English and Spanish support

## 🚀 How to Run the App

### Prerequisites
Make sure you have these installed on your computer:
- [Node.js](https://nodejs.org) (v18 or higher)
- [Expo Go](https://expo.dev/client) app on your phone (iOS or Android)

### Step 1 — Clone the Repository
```bash
git clone https://github.com/Vayala13/practice-repo-making.git
cd practice-repo-making
git checkout web-app-work-Nate
cd fmla-app
```

### Step 2 — Install Dependencies

## 🛠️ Tech Stack
- **React Native** — Mobile framework
- **Expo** — Development platform
- **Expo Router** — File-based navigation

- **TypeScript** — Type safety

---

## ⚠️ Important Notes
- Never commit your `.env` file or share your API key publicly
- The AI chat provides **general information only** and is **not legal advice**
- FMLA information is based on federal law — state laws may vary
- Always refer users to HR or an employment attorney for specific situations
